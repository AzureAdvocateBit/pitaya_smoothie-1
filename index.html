A dark VS Code theme inspired by the Ultraviolet Pantone Color of the year 2018 and the Outrun aesthetics!

To use this theme:

  1. Install VS Code Studio
  2. Open the extension sidebar on VS Code and search for Pitaya Smoothie
  3. Click Install
  4. Choose the theme using Preferences -> color theme

If you encounter any bugs or problems please open an issue on the project's repository.
